Eggy is not just a timer; it is a high-fidelity laboratory for molecular gastronomy, designed for the most discerning home scientists and professional chefs. It combines advanced thermodynamics with a breathtaking "Anti-Gravity" UI to ensure every egg—from the tiny Quail to the massive Ostrich—is cooked to molecular perfection.

---

## 🔬 Core Technologies

### 🧬 Molecular Physics Engine
Eggy operates at a strict **100°C laboratory baseline**. Our custom physics engine accounts for:
- **Species-Specific Thermal Conductivity**: Unique algorithms for Hen, Duck, Goose, Emu, Ostrich, and more.
- **Altitude Correction**: Real-time atmospheric pressure calculations to adjust boiling points.
- **Protein Phase Tracking**: Visualizing the transition from liquid to the perfect jammy "Sun Side" state.

### 🐣 Avian Biodiversity Cabinet
Explore the diverse world of avian specs with our geometrically accurate cabinet.
- **Logarithmic Scaling**: Experience the relative dominance of exotic eggs while maintaining UI harmony.
- **Characteristic Realism**: Procedural textures (like Quail mottling) and characteristic colors ( Oceanic Teal Emu shells).
- **Anti-Gravity UX**: A seamless, glassmorphic carousel that centers itself dynamically across all screen sizes.

### 🧠 Gemini-Powered AI Sous-Chef
Deeply integrated with Google's Gemini Pro, the Eggy assistant offers:
- **Implicit Context Recognition**: Eggy remembers your previous queries, supporting natural, pronoun-heavy dialogue.
- **Scientific Research Capabilities**: Real-time browsing and domain-aware culinary insights.
- **Mascot Interaction**: A 3D claymation companion that emotes based on your cooking progress.

## 🚀 Getting Started

### Prerequisites
- [Flutter SDK](https://docs.flutter.dev/get-started/install) (Stable)
- [OpenRouter](https://openrouter.ai/) or Google Gemini API Key

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/gwaisey/eggy.git
   ```
2. Create a `.env` file in the root directory:
   ```text
   GEMINI_API_KEY=your_key_here
   ```
3. Install dependencies:
   ```bash
   flutter pub get
   ```
4. Run the application:
   ```bash
   flutter run
   ```
